# Public Assets

This directory stores public website assets that are safe to publish with the repository.

## Naming

- Placeholder graphics: `assets/placeholders/page-<page>-<slot>.svg`
- Future real media: `assets/media/page-<page>-<slot>.<ext>`

Recommended future media formats:

- Photos: `.webp`
- Graphics / diagrams: `.svg`

## Current placeholder slots

- `page-home-case-factory`
- `page-home-case-campus`
- `page-home-case-community`
- `page-contact-qr`
- `page-404-hero`

## Replacement rule

When real assets are ready, replace the placeholder `<img>` references in the corresponding page file with the final media file path under `assets/media/`.
